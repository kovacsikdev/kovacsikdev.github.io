Web app (React js) displaying real time data of vehicle location and status on Google Maps. I implemented 20 bots to accurately simulate multiple vehicles in the real world updating their location and vehicle status. The bots individually stream their data to my custom server (Node js), which in turn streams vehicle updates to any connected clients. All data streaming utilizes Web Sockets. The overall architecure of what I did can be found in the web app.

Features:
  • Display all 20 bots simultaneously updating their location in a frictionless manner
  • Sortable list of all connected bots
  • Selecting a vehicle will:

    • Display current engine status
    • Individually display tire pressure with an animated chart made in d3.js
    • Display a simplified, fully 3D vehicle showing the status of all health indicators made in three.js
